回退命令：


回退到上个版本
---------------------------------------------------
1) $ git reset --hard HEAD^       // 回退到上个版本
2) $ git push origin dev -f       // 强推到远程 


回退到N次提交之前
-----------------------------------------------------------
1）$ git reset --hard HEAD~3        // 回退到前3次提交之前，以此类推，回退到n次提交之前

2）$ git push origin HEAD --force   // 强推到远程


回退到某次提交的版本上
-----------------------------------------------------------------------------------
$ git reset --hard commit_id     // 退到/进到 指定commit的sha码
2) $ git push origin dev -f      // 强推到远程dev
